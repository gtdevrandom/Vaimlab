# GUIDE DES ARMES (VALORANT)

## 1. PISTOLETS (SIDEARMS)

### Classic
- Chargeur : 12
- Cadence : 6.75 tirs/s
- Reload : ~1.75–2.5 s
- Recoil : très faible (premiers tirs précis, burst clic droit dispersé)

### Shorty
- Chargeur : 2
- Reload : ~1.75 s
- Recoil : quasi nul (shotgun → spread fixe)

### Frenzy
- Chargeur : 13
- Cadence : 10 tirs/s
- Reload : ~1.5 s
- Recoil : élevé en spray (vertical rapide)

### Ghost
- Chargeur : 15
- Reload : ~1.5 s
- Recoil : faible → très bon en tap

### Sheriff
- Chargeur : 6
- Cadence : 4 tirs/s
- Reload : ~2.25 s
- Recoil : très élevé → reset lent entre tirs


## 2. SMG

### Stinger
- Chargeur : 20
- Cadence : 16 tirs/s
- Reload : ~2.25 s
- Recoil : très violent (spray incontrôlable après 6–8 balles)

### Spectre
- Chargeur : 30
- Cadence : 13.33 tirs/s
- Reload : ~2.25 s
- Recoil : modéré, facile à contrôler en spray court


## 3. FUSILS (RIFLES)

### Bulldog
- Chargeur : 24
- Cadence : ~9.14 tirs/s
- Reload : 2.5 s
- Recoil : moyen, burst ADS très stable

### Guardian
- Chargeur : 12
- Cadence : 5.25 tirs/s
- Reload : 2.5 s
- Recoil : fort mais reset rapide → tap uniquement

### Phantom
- Chargeur : 30
- Cadence : 11 tirs/s
- Reload : 2.5 s
- Recoil :
  vertical + léger horizontal
  spray contrôlable
  silencieux → pas de tracers

### Vandal
- Chargeur : 25
- Cadence : 9.75 tirs/s
- Reload : 2.5 s
- Recoil :
  plus dur que Phantom
  pattern vertical + random horizontal
  one-tap à toutes distances


## 4. SNIPERS

### Marshal
- Chargeur : 5
- Cadence : 1.5 tirs/s
- Reload : 2.5 s
- Recoil : quasi nul (bolt-action)

### Outlaw
- Chargeur : 2 (10 réserve)
- Reload : ~3.8 s
- Recoil : élevé entre les 2 tirs

### Operator
- Chargeur : 5
- Cadence : 0.6 tirs/s
- Reload : ~3.7 s
- Recoil : énorme → reposition obligatoire


## 5. SHOTGUNS

### Bucky
- Chargeur : 5
- Reload : ~2.5 s
- Recoil : faible mais dispersion élevée

### Judge
- Chargeur : 7
- Cadence : ~3.5 tirs/s
- Reload : ~2.2 s
- Recoil : fort en spam


## 6. MITRAILLEUSES (HEAVY)

### Ares
- Chargeur : 50
- Reload : ~3.25 s
- Recoil : devient plus stable en spray

### Odin
- Chargeur : 100
- Reload : ~5 s
- Recoil :
  énorme au début
  puis stabilisation progressive
